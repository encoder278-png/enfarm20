# EnFarm AI — Swahili Agricultural Assistant for Tanzanian Farmers

EnFarm is an AI-powered agricultural assistant that helps Tanzanian smallholder farmers diagnose crop diseases, get weather forecasts, and receive farming advice — all in Swahili.

## Problem
Millions of Tanzanian maize farmers lose crops every season to diseases they cannot identify. Agricultural extension officers are scarce. Farmers need immediate, accurate, affordable help.

## Solution
EnFarm uses AI to give farmers instant crop disease diagnosis and farming advice in Swahili through a simple web interface — no agricultural expertise required.

## Current Features
- Swahili AI farming assistant powered by Gemini
- Crop image analysis and disease diagnosis
- Farmer memory via Firestore

## Tech Stack
- Google AI Studio
- Gemini API
- Firebase Firestore
- Render (backend deployment)

## Target Users
Smallholder maize farmers in Tanzania

## Status
Active development — MVP in progress

## Developer
